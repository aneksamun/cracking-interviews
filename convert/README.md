### Convert Tasks

#### Digits to words

Displays given number in words, for example, 89 is displayed as "eight nine"

#### List to integers

Converts the list of the integers into a single digit and vice versa.
```
[1,2,3] -> 123
123 -> [1,2,3]
```
#### Decimals to Binary

Converts given decimal to its binary representation
```
2 -> 10
4 -> 100
7 -> 111
```
