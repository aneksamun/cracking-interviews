# Cracking Interviews

Contains algorithms and solutions for the different problems.

### How to run (Unix)?

- C++
```
g++ -c <filename>.cpp --std=c++17
g++ -o <prog_name> <filename>.o
./<prog_name>
```
- Haskell
```
ghci <filename>.hs
or
ghci
:load <filename>.hs
```
- Clojure
```
clojure
(load-file "<filename>.clj")
```
